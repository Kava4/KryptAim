# Safety features

Humanize recoil output so patterns feel less mechanical.

---

## Randomization

- **Enable randomisation** — adds jitter to movement.
- **Random strength** — scale of jitter (default ~5).

Useful when fine-tuning “natural” feel without changing the core pattern shape.

---

## Global multipliers

| Control | Effect |
|---------|--------|
| **X control** | Scale horizontal compensation (%) |
| **Y control** | Scale vertical compensation (%) |
| **Speed control** | Scale timing / speed of steps (%) |

These apply on top of pattern values in Simple, Advanced, and Game Engine modes.
