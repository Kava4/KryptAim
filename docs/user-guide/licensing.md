# Access mode

AimSync is currently running in **free access mode**.

## Current behavior

- All main features are unlocked, including **Game Engine** and **cloud upload**.
- The dashboard shows `FREE ACCESS ENABLED` in Global Settings.
- License/trial checks are intentionally bypassed for this release.

## Placeholder for future licensing

The app keeps internal placeholder routes and UI slots for licensing so paid/trial access can be re-enabled in a future update without redesigning the dashboard.
