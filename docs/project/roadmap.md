# Roadmap

Planned and completed work for AimSync. Status reflects the repository as of the current development cycle.

## Completed

- Input method selection (hardware vs software)
- Cloud pattern repository (Vercel + Supabase)
- Public landing page on Vercel
- Supporter recognition in dashboard
- Premium license verification and HWID binding
- Anti-tamper premium cache
- Free trial (2 hours active time per day)
- Multi-game weapon modules (CS2, Valorant, PUBG, R6S, Rust)

## In progress

- Payment automation (Ko-fi webhook → instant license activation)

## Planned

| Feature | Description |
|---------|-------------|
| Game integration | Broader title support beyond current modules |
| Weapon database | More CS2 weapons and refined sprays |
| Dynamic profile switching | Detect active game process and switch config |
| Humanized input curves | Bézier / spline movement paths |
| HUD overlay | On-screen status and active weapon |
| Auto-weapon detection | OCR-based weapon switching |
| State-based scaling | Different multipliers per stance (crouch / stand) |
| macOS | Quartz input + driver work + permissions model |

## How priorities are chosen

Hardware availability, community demand, and maintenance cost drive ordering. Items may move or split into smaller releases.
