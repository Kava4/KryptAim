# Simple recoil

Fast vertical/horizontal compensation without building a full pattern.

---

## Controls

| Slider | Purpose |
|--------|---------|
| **X** | Horizontal pull per tick |
| **Y** | Vertical pull per tick |
| **Delay** | Milliseconds between steps |

---

## Usage

1. Enable master switch in [Global settings](Global-Settings).
2. Set **Simple Recoil** values.
3. Hold toggle hotkey in-game.

Use **Reset** to return sliders to defaults.

---

## When to use Advanced / Game Engine

- **Recoil Lab** — custom multi-step sprays, save/load patterns.
- **Game Engine** — weapon-specific CS2 profiles scaled to sensitivity.
